Arduino Piano

A full octave keyboard built using capacitors, Arduino Uno and speakers and tested device with function generator and oscilloscope. It uses the Processing function on Arduino IDE to create features that record, playback and teach users how to play the piano.

Elixr: Empowering mental health patients to reach their goals

Psychotherapy is one of the most successful treatments for mental health and if patients remain engaged with their treatment, it leads to faster and longer-lasting results. However, statistics show that up to 50% of patients don't adhere to their treatment outside of the therapy session.

Elixr is a solution to help mental health patients complete their treatment. It is a platform for therapists and their patient to collaborate and create a customized treatment plan during therapy sessions. And from home, patients have access to assigned resources and activities through their own device. Gamification elements and strategic user flows based on clinical research address the factors that frequently prevented treatment compliance.

Re-tail

A chrome extention to track retail prices on any e-commerce website so users can make smart purchases. Get daily updates on price changes and know when it's the right time to buy.

Unlike other retail chrome extensions, all user data is saved locally and nothing is stored on servers and sold to third-parties.

Device Tracking App

Overview

A discreet tracking app for Android devices or any browser. It sends periodic location updates via email.

User Features

  • Android app looks like a system feature and will not alert the user
  • Initial setup may require user permission to access location
  • Default interval for email updates: every 30 minutes
  • Android app: Marshmallow or above
  • Browsers: Chrome, Explorer, Firefox. Safari, Opera

Technology

Android app and web app were written in Java and JavaScript respectively. Backend was written in Node.js with Express and the server is hosted on Heroku.

Dance Wearable: Music through movement

A proof of concept wearable to control music properties of a song through dance moves. Using an Arduino Midi, gyroscope and accelerometer, vertical and horizontal movement can alter pitch, speed, vibrato and volume of the music.
